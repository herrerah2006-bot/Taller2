# Taller 2 - Backend REST API con Spring Boot

Proyecto desarrollado como parte del taller práctico de Software Development en ITSQMET. El objetivo principal fue la implementación de una API REST para la gestión de datos clínicos (Médicos, Pacientes y Citas) utilizando Java y Spring Boot.

## Tecnologías Utilizadas
* **Lenguaje:** Java
* **Framework:** Spring Boot
* **Base de Datos:** MySQL
* **Gestión de BD:** MySQL Workbench
* **Testing:** Postman (Validación de endpoints REST)
* **Entorno de Desarrollo:** VS Code

## Estructura del Proyecto
El backend expone endpoints funcionales bajo el protocolo HTTP, permitiendo las operaciones fundamentales de un CRUD (Create, Read, Update, Delete) para la entidad clínica gestionada.

## Configuración y Ejecución
1. Clonar el repositorio.
2. Configurar la conexión a la base de datos MySQL en el archivo `application.properties`.
3. Ejecutar el proyecto mediante Spring Boot.
4. Utilizar Postman para realizar peticiones `GET` y `POST` hacia los endpoints configurados.
